### 6.2.2

Repeat exercise 6.2.1 for the following assignment statements

1. a = b[i] + c[j]
2. a[i] = b\*c - b\*d
3. x = f(y+1) + 2
4. x = \*p + &y

#### Answer

1. a = b[i] + c[j]

- Quaternary formula

0) =[] b i t1
1) =[] c j t2
2) + t1 t2 t3
3) = t3 a

- ternary formula

0) =[] b i
1) =[] c j
2) + (0) (1)
3) = a (2)

- Indirect ternary

0) =[] b i
1) =[] c j
2) + (0) (1)
3) = a (2)

0)
1)
2)
3)

2. a[i] = b\*c - b\*d

- Quaternary formula

0) * b c t1
1) * b d t2
2) - t1 t2 t3
3) []= a i t4
4) = t3 t4

- ternary formula

0) * b c
1) * b d
2) - (0) (1)
3) []= a i
4) = (3) (2)

- Indirect ternary

0) * b c
1) * b d
2) - (0) (1)
3) []= a i
4) = (3) (2)

0)
1)
2)
3)
4)

3. x = f(y+1) + 2

- Quaternary formula

0) + y 1 t1
1) param t1
2) call f 1 t2
3) + t2 2 t3
4) = t3x

- ternary formula

0) +        y     1
1) param    (0)
2) call     f     1
3) +        (2)   2
4) =        x     (3)

- Indirect ternary

0) + y 1
1)param(0)
2) call f 1
3) + (2) 2
4) = x(3)
