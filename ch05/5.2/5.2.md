### 5.2.3

Suppose we have a production A -> BCD.The four non-terminal symbols A, B, C and D all have two attributes, the synthetic attribute s and the inherited attribute i.For each set of rules below, indicate (1) whether these rules meet the requirements defined by the S attribute (2) whether these rules meet the requirements defined by the L attribute (3) whether there is an evaluation process consistent with these rules?

1. A.s = B.i + C.s
2. A.s = B.i + C.s , D.i = A.i + B.s
3. A.s = B.s + D.s
4. ! A.s = D.i , B.i = A.s + C.s , C.i = B.s , D.i = B.i + C.i

#### Answer

1. 否， ？   [No, Yes, follows an evaluation order since there are no cycles in the dependancy graph]
2. 否， 是   [No, Yes, follows an evaluation order since no cycles]
3. 是， 是   [Yes, Yes, also follows evaluation order sicne no cycles, the evaluation order is the topological sort]
4. 否， 否   [No, No, DOESN'T follow any evaluation order since there is a cycle. D.i depends on B.i and C.i but B.i depends on A.s which in turn depends on D.i so there can't be any topological sort.]
