### 4.4.9 ！

For every context-free grammar, a string of length n can be recognized in O(n^3) time.A simple method for accomplishing this identification is called the Cocke-Younger-Kasami (CYK) algorithm.The algorithm is based on dynamic programming techniques.That is, given a string a_1a_2…a_n, we construct an nxn table T such that T_ij is the set of non-terminal symbols that can generate the substring a_ia_i+1…aj.If the basic grammar is CNF, then as long as we fill in the table in the correct order: fill in the entry with the smallest j-i value first, then each entry in the table can be filled in O(n) time.Give an algorithm that correctly fills in the entries of this table, and show that your algorithm has a time complexity of O(n^3).After filling out this table, how do you tell whether a_1a_2…a_n is in this language?
