### 4.6.5

Explain the following grammar

S->YesAb|BbBa
A−>ε
B −>ε

It's LL(1), but not SLR(1).

#### Answer

1. The grammar is LL(1)

See Section 4.4.3, Judgment Criteria on p142

2. The grammar is not SLR(1)

I_0

S' -> .S
S -> .AaAb
S -> .BbBa
A -> .
B -> .

Since FOLLOW(A) = FOLLOW(B) = [a, b], a specification violation occurs when the input after I_0 is a or b.

### 4.6.6

Explain the following grammar

S->SA|A
A->a

It is SLR(1), but not LL(1)

#### Answer

1. The grammar is not LL(1)

Both `S -> SA` and `S -> A` can deduce the string starting with a, so it is not LL(1)

2. The grammar is SLR(1)

The syntax analysis table generated by this grammar has no conflicts.
