### 2.8.1

The for statement in C language and JAVA language has the following form:

for(expr1; expr2; expr3) stmt

The first expression is executed before the loop and is usually used to initialize the loop index.The second expression is a test that is executed before each iteration of the loop.If the result of this expression becomes 0, exit the loop.The loop province can be viewed as the statement {stmt expr3}.The third expression is executed at the end of each iteration and is typically used to increment the loop index.So the meaning of the for statement is similar to:

expr1; while(expr2){stmt expr3;}
