other -> [bcdfghjklmnpqrstvwxyz]

2.

3.

\/\*([^*"]*|".*"|\*+[^/])*\*\/

The contents appearing between /* and */ are closures in three different situations (strings of any length randomly combined):

1. `[^*"]*`: strings of any length with all symbols except * and ".
2. `".*"`: A string enclosed by two quotation marks, within which any symbol is allowed.
3. `\*+[^/]`: When * appears, it is followed by a symbol other than /.

This expression also matches the case of `/**/`.

It should be noted that this expression implies the following constraints:

- cannot contain unpaired quotes
- Single quotes are not considered (in some languages, single quotes also represent strings)

### 3.3.9!

Try to prove: for every expression containing the form r{m, n}, there is an equivalent regular expression that does not contain the repetition operator.

#### Answer

r{m,n} is equivalent to r.(m items).r | r.(m+1 items).r | ... | r.(n items).r
