### 3.6.1 ！

Figure 3-19 in the exercise in Section 3.4 calculates the failure function of the KMP algorithm.Show how to construct a DFA with n+1 states from the known keywords b_1b_2…b_n that recognizes the language .*b_1b_2…b_n , given a known failure function.Going a step further, show that the time complexity of constructing this DFA is O(n).

#### Answer

For convenience of explanation, taking the string abbaabb in 3.4.3-3 as an example, the failure function is:

- n   : 1, 2, 3, 4, 5, 6, 7
- f(n): 0, 0, 0, 1, 1, 2, 3

The constructed DFA is:

![3 6 1](https://f.cloud.github.com/assets/340282/441972/36094510-b130-11e2-8e22-14aba49e8213.gif)

The pseudocode to construct this DFA is:

for (i = 0; i< n; i ++) {
move[s[i], c] = {
if ( c == b_1b_2…b_n[i] ) {
goto s[i+1]
} else {
goto s[f(i)]
}
}
}

Obviously, given f(n), the time complexity of this algorithm is O(n)
