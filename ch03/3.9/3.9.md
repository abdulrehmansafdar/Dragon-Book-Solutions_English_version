- Calculation steps

The firstpos value of the root node of this tree is {1, 2, 3}, so the starting state of D is this set, let A = {1, 2, 3}.Compute Dtran[A, a] and Dtran[A, b], where position 1 corresponds to a and position 2 corresponds to b.Therefore Dtran[A, a] = followpos(1) = {1, 2, 3}, Dtran[A, b] = followpos(2) = {1, 2, 3}.Both results are set A. This calculation does not produce a new state and the calculation ends.
