### 3.8.3 ！

Suppose we modify the definition of DFA so that each state has zero or one transition on each input symbol (instead of exactly one transition as in the standard DFA definition).Then, some regular expressions can have a smaller "DFA" than the DFA that would result from constructing it according to the standard definition.Give an example of such a regular expression.

#### Answer

Taking the language defined by the regular expression ab as an example, assume that the input character set of this language is {a, b}

Standard DFA

![3 8 3-1](https://f.cloud.github.com/assets/340282/451932/881521e6-b2c3-11e2-935c-0a6c8177ad87.gif)

Corrected DFA

![3 8 3-2](https://f.cloud.github.com/assets/340282/448895/11fa403a-b260-11e2-9f30-f6c2b6c72539.gif)

Compared to a standard DFA, there are 0 transitions on b in state 0, 0 transitions on a in state 1, and 0 transitions on a and 0 transitions on b in state 2.Obviously, the DFA obtained according to the revised definition is smaller.
